Code of Conduct
1. Respect for Original Licensing
Users are strictly prohibited from removing, modifying, or concealing licensing information included in this project. Any modifications to the source code must retain the original license provided.

2. Prohibition on Misuse for Profit
It is prohibited to use the modified version of this project for personal or commercial profit if such actions violate the terms of the original license.

3. Transparency in Modifications
If you modify or redistribute this code, you are required to:

Include the original license without changes.
Provide proper attribution to the original contributors.
Provide a changelog or record of changes if requested.
4. Violation of Conduct
If a violation is identified, such as removing the license for commercial gain, the following actions may be taken:

Reporting the violation to GitHub or the relevant platform.
Legal action in accordance with applicable copyright laws.
